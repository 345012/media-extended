# 创建您的首个媒体笔记

欢迎您开启使用 Media Extended 进行高效视频笔记记录。我们已经简化了操作流程，即便您是初次尝试，也能轻松开始记录视频笔记。

## 打开播放器 [#open-player]

首先，我们试着在 Obsidian 中直接打开一个 bilibili 视频。只需点击 [这个视频的链接](obsidian://mx-open?url=https%3A%2F%2Fwww.bilibili.com%2Fvideo%2FBV1GJ411x7h7) 即可。

## 记笔记 [#taking-notes]

在插件提供的升级版播放器中打开视频后，您会看到播放器顶端有两个关键按钮：**截图** 和 **时间戳**。

- **时间戳&nbsp;⭐&nbsp;**：点击此按钮会在 Obsidian 中创建一份新笔记，其中包含了一个直接指向您正在观看视频某刻的时间戳。
- **截图&nbsp;📷&nbsp;**：通过这个按钮，您可以捕获当前视频帧的截图，并将其作为图像文件保存到您的资料库中，同时包含时间戳和视频链接。

准备好开始做笔记时，只需点击 **时间戳** 或 **截图** 按钮，即可捕捉您的想法。

<Bleed>
![首个媒体笔记](./first-note.png)
</Bleed>

想要在做笔记时进一步控制播放器，可以打开命令面板，搜索 `Media Extended:` 查看所有可用命令。这里您可以快进、回放、暂停、播放等操作。根据 obsidian 的插件规范，默认情况下不设置快捷键，但您可以遵循 [此指南](https://publish.obsidian.md/help-zh/%E4%B8%AA%E6%80%A7%E5%8C%96%E5%AE%9A%E5%88%B6/%E8%87%AA%E5%AE%9A%E4%B9%89%E5%BF%AB%E6%8D%B7%E9%94%AE) 自行设置快捷键。

<Bleed>
![媒体命令](./media-commands.png)
</Bleed>

你可以在 [命令合集](/reference/commands) 中找到所有命令及其描述。

<Callout type="info">

    您可能会注意到播放器现在在标签页顶部以 🟥 红色图标标记，这表示该播放器处于“活跃”状态，正在被用于记录笔记。这在您在其他不是“Media Note”的笔记内工作时特别有用。

    如果你希望 v3 以前旧版的 ["Link with pane"](https://github.com/PKM-er/media-extended/wiki/Open-Links-in-One-Shared-Player) 功能，你也可以在任意活跃的笔记上使用上述的命令或你自行分配的快捷键来控制播放器。

</Callout>



## 共享播放器 [#shared-player]

当您需要处理含有不同媒体的多份笔记时，频繁切换播放器可能会很烦人。Media Extended 通过与 Obsidian 的“固定标签页”功能集成，巧妙解决了这一问题。只需右键点击播放器标签并选择“固定”，即可实现播放器跨笔记共享，使您能够在不同笔记间自如切换，避免播放器堆积。

<Bleed>
![固定播放器](./pin-player.png)
</Bleed>

## 从多种来源打开媒体 [#different-sources]

import { Tabs, Callout } from 'nextra/components'

下面将列举几种常见的媒体来源，以及如何在 Media Extended 中打开它们：

<Tabs items={['本地文件', '网页']}>
    <Tabs.Tab>
    1. 对于存储在您的资料库中的媒体，你不需要学习额外的操作。插件会自动识别并在增强播放器中打开它。
    2. 对于计算机上的其他文件，可以通过 [媒体快速切换器](#quick-switcher) 来打开。你可以直接输入路径，也可以通过 `Open local file` 选项使用文件选择器。
    </Tabs.Tab>
    <Tabs.Tab>
    无论是使用自建媒体服务器、NAS 还是公共媒体托管服务（如 YouTube、Vimeo、bilibili），只要直接 URL 以文件扩展名结束或可获取包含媒体的网页，即可在增强播放器中打开媒体。
    
    1. 在 [媒体快速切换器](#quick-switcher) 中输入 URL 或媒体 ID（适用于 YouTube 和 bilibili）并按回车
    2. 在浏览器中，将以下前缀添加到任意 URL 前并回车：
    
    ```txt copy
    obsidian://mx-open/
    ```

    <Callout type="warning">
        如果网站要求登录才能访问内容或在播放期间请求登录，您可以通过插件设置中的浏览器页面登录，或使用 `Login website` 命令。
        
        对于 bilibili，这是必须的，因为它会在播放时中断并请求登录。

        <Bleed>
        ![login from setting tab](./login-setting.png)
        </Bleed>
    </Callout>
    <Callout type="warning">
        若您的媒体需要 DRM，其将无法正常播放，因为 obsidian.md 不支持 DRM
    </Callout>
    </Tabs.Tab>
</Tabs>

### 媒体快速切换器 [#quick-switcher]

若要打开不在您资料库中的媒体文件，可以使用媒体快速切换器。通过点击工具栏的播放按钮或使用命令面板搜索 `Media Extended: Open media` 来调用此功能。
